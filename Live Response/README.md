## Live Response scripts for MDE.

The purpose of these PowerShell scripts is to be able to remotely collect forenisc information, search for files, delete files/folder, and collect files into a password protected ZIP files. All the collector scripts were written to able to be deployed via Microsoft Defender for Endpoint (MDE) aka Microsoft 365 Portal, remotely (PSEXEC, PSSession, etc...), and by an administrator on a local system.

All PowerShell scripts are built as a one-stop shop script that have all the necessary external tools contained within a ZIP file that is base64 encoded and embedded within the script.
